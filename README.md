COPPY CAMSS
영업관리 프로그램 CAMSS를 복사한 프로젝트입니다.

## 구성

BACKEND

```bash
framework : express.js
db : mongoDB or nodejs sequelize
```

```bash
framework : next.js
library : redux, redux-toolkit, redux-saga
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
